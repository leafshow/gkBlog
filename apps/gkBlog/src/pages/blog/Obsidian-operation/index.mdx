## Obsidian 一款强大的笔记管理软件

### 一、软件基础操作

1. Obsidian双向链接的基本用法

```bash
链接到某一篇笔记：[[ ]]
链接到某一篇笔记中的某个标题：[[ # ]]
链接到某一篇笔记中的某个段落（块）：[[ # ^ ]]
为链接创建定义（关键词）：[[ | 关键词]]
链接到外部文件如印象笔记：[关键词]（链接）
```

7.  Image auto upload
    如果你把所有的笔记配图都保存在 Obsidian 资料库中，那么，繁多的附件必定让你心烦意乱。

所以，我的选择是：所有的配图，都以图床的方式保存。 像这样

这样一来，所有的附件，都只是一个个链接。首先，链接无需管理；其次，如果我要把笔记从 Obsidian 迁移到其它工具中，也只需要迁移文档即可，跟图片无关。
而 Image auto upload 这款插件，便是在 Obsidian 实现图片自动上传的。
这款插件需要配合软件 Picgo ，之后，使用 Obsidian 上传图片时，我可以将图片拖动进来，或者直接粘贴进来，它都会上传到预先配置好的图床中。
我用这样的方式，只花了不到半小时的时间，就完成了把数十万字的文档，从 Bear 迁移到 Obsidian 的全过程。

9.  Export Image

10. Better Export PDF
    众所周知，Obsidian原生提供了将笔记导出为PDF的功能。

但这一功能，有不少的缺陷，其中最被人诟病的，便是导出的PDF中，没有目录，即便你在笔记中输入了多级标题。

而 Better Export PDF 这款插件，就解决了这一问题。

安装插件之后，打开需要导出的笔记，在命令面板中，选择”Better Export PDF“，便能够导出更加完善的PDF文件：

12. Custom Image Auto Uploader
    您可以将笔记中的图片上传保存到您的远端服务器、家庭 NAS 或者同步保存在您的云存储上（阿里云 OSS 、亚马逊 S3 、Cloudflare R2 ）,并且您还可以对图片进行 拉伸裁剪以及修改尺寸。
    <Image
      src="https://pic3.zhimg.com/v2-a27d84ff2eac8b65a0c5f8e7c5ce5976_b.webp"
      alt="图片描述"
      width={800}

      isArticleImage={true}
    />
    插件功能:

- 批量下载笔记内网络图片到本地
- 批量上传笔记中本地图片到远端服务 ( Image Api Gateway )
- 拖拽,粘贴 上传图片
- 批量将笔记内图片保存到 远端服务器，例如您的Web服务器或家庭NAS
- 批量将笔记内图片保存到云存储内，例如阿里云 OSS 、亚马逊 S3 、Cloudflare R2 。
- 支持多端使用, 支持 Windows, MacOS, Linux, Android, iOS
- 支持多国语言
- 增加笔记属性或正文内图片拉伸裁剪功能 (例如 对 hugo,hexo 标题图 cover 处理)
  - 等比左上填充(裁剪)
  - 等比居中填充(裁剪)
  - 固定尺寸拉伸
  - 等比适应
