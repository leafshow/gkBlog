## VidHub 破解激活注册

1. Apple商店下载 vidhub
2. 打开一下VidHub APP
3. Mac终端执行指令：

```bash
defaults write ~/Library/Containers/com.mac.utility.media.hub/Data/Library/Preferences/com.mac.utility.media.hub.plist IsOneTimePayment -bool true
```
